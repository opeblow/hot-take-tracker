# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by opening a GitHub Security Advisory at:

https://github.com/opeblow/hot-take-tracker/security/advisories/new

Do not disclose the issue publicly until it has been addressed.

## Scope

- API key exposure in logs or error messages
- Injection vulnerabilities in the statement endpoint
- Cross-origin data leakage
- Dependency vulnerabilities with known CVEs

## Response

You will receive a response within 72 hours acknowledging the report. A fix timeline will be provided based on severity.
